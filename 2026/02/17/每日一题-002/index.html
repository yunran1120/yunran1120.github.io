题目：

已知微分方程 y′′−2(1+tan⁡2x)y=0y'' - 2(1+\tan^2x)y = 0y′′−2(1+tan2x)y=0 有一个解 y1=tan⁡xy_1 = \tan xy1​=tanx，求其通解。

[!NOTE]
关键点：刘维尔公式 常数变易法

[!TIP]
本文中验证两特解线性无关的方法为朗斯基行列式（Wronskian Determinant），更加直观与明显。

在日常计算时，也可选用比例法，即检查是否存在一个常数 kkk，使得 y1(x)y2(x)=k\frac{y_1(x)}{y_2(x)} = ky2​(x)y1​(x)​=k：

  • 如果比值是常数，则两个函数线性相关。
  • 如果比值依赖于 xxx（不是常数），则两个函数线性无关。

示例：e2xex=ex\frac{e^{2x}}{e^x} = e^xexe2x​=ex 结果随 xxx 变化，因此两个函数线性无关。

PROOF1:（使用刘维尔公式）

步骤1：将方程化为标准形式

二阶线性齐次微分方程的通用标准形式为：

y′′+P(x)y′+Q(x)y=0y'' + P(x)y' + Q(x)y = 0 y′′+P(x)y′+Q(x)y=0

对比原方程 y′′−2(1+tan⁡2x)y=0y'' - 2(1+\tan^2x)y = 0y′′−2(1+tan2x)y=0，无 y′y'y′ 项，因此对应系数为：

P(x)=0,Q(x)=−2(1+tan⁡2x)P(x) = 0,\quad Q(x) = -2(1+\tan^2x) P(x)=0,Q(x)=−2(1+tan2x)

步骤2：刘维尔降阶公式（二阶方程专用）

对于二阶线性齐次方程，若已知一个非零特解 y1(x)y_1(x)y1​(x)，则第二个与它线性无关的特解，可由刘维尔公式直接推导：

y2(x)=y1(x)⋅∫e−∫P(x)dxy12(x) dxy_2(x) = y_1(x) \cdot \int \frac{e^{-\int P(x)dx}}{y_1^2(x)} \, dx y2​(x)=y1​(x)⋅∫y12​(x)e−∫P(x)dx​dx

公式原理：基于刘维尔定理，朗斯基行列式满足 W(x)=W(x0)e−∫x0xP(t)dtW(x)=W(x_0)e^{-\int_{x_0}^x P(t)dt}W(x)=W(x0​)e−∫x0​x​P(t)dt，以此实现降阶，快速求解第二个特解。

步骤3：代入特解计算第二个线性无关解

已知特解 y1(x)=tan⁡xy_1(x)=\tan xy1​(x)=tanx，且 P(x)=0P(x)=0P(x)=0，分步计算：

  1. 计算指数项

    ∫P(x)dx=∫0 dx=0,e−∫P(x)dx=e0=1\int P(x)dx = \int 0 \, dx = 0,\quad e^{-\int P(x)dx}=e^0=1 ∫P(x)dx=∫0dx=0,e−∫P(x)dx=e0=1

  2. 代入刘维尔降阶公式

    y2(x)=tan⁡x⋅∫1tan⁡2x dx=tan⁡x⋅∫cot⁡2x dxy_2(x) = \tan x \cdot \int \frac{1}{\tan^2 x} \, dx = \tan x \cdot \int \cot^2 x \, dx y2​(x)=tanx⋅∫tan2x1​dx=tanx⋅∫cot2xdx

  3. 计算三角积分（利用恒等式 cot⁡2x=csc⁡2x−1\cot^2 x = \csc^2 x - 1cot2x=csc2x−1）

    ∫cot⁡2x dx=∫(csc⁡2x−1) dx=−cot⁡x−x+C\int \cot^2 x \, dx = \int (\csc^2 x - 1) \, dx = -\cot x - x + C ∫cot2xdx=∫(csc2x−1)dx=−cotx−x+C

  4. 取积分常数 C=0C=0C=0（仅需一个特解，常数可吸收到通解的任意常数中），代入化简

    y2(x)=tan⁡x⋅(−cot⁡x−x)=−1−xtan⁡xy_2(x) = \tan x \cdot (-\cot x - x) = -1 - x\tan x y2​(x)=tanx⋅(−cotx−x)=−1−xtanx

步骤4：验证线性无关性（朗斯基行列式）

两个函数线性无关的充要条件：其朗斯基行列式在定义域内不恒为0。
朗斯基行列式定义：

W(y1,y2)=∣y1y2y1′y2′∣W(y_1,y_2) = \begin{vmatrix} y_1 & y_2 \\ y_1' & y_2' \end{vmatrix} W(y1​,y2​)=​y1​y1′​​y2​y2′​​​

先求导：

y1′=sec⁡2x=1+tan⁡2x,y2′=−tan⁡x−xsec⁡2xy_1' = \sec^2 x = 1 + \tan^2 x,\quad y_2' = -\tan x - x\sec^2 x y1′​=sec2x=1+tan2x,y2′​=−tanx−xsec2x

朗斯基行列式在定义域内恒不为0，因此 y1y_1y1​ 与 y2y_2y2​ 线性无关。

利用"刘维尔定理"解题的思路：

刘维尔公式的核心价值，是解决「二阶线性齐次微分方程，已知一个非零特解，求第二个线性无关特解，最终得到通解」的问题。

二阶线性齐次微分方程的通解结构定理，是所有操作的前提：

对于标准形式的二阶线性齐次方程：

y′′+P(x)y′+Q(x)y=0y'' + P(x)y' + Q(x)y = 0 y′′+P(x)y′+Q(x)y=0

若 y1(x)y_1(x)y1​(x) 和 y2(x)y_2(x)y2​(x) 是方程的两个线性无关的特解（即不成比例），则方程的通解为：

y=C1y1(x)+C2y2(x)(C1,C2为任意常数)y = C_1 y_1(x) + C_2 y_2(x) \quad (C_1,C_2\text{为任意常数}) y=C1​y1​(x)+C2​y2​(x)(C1​,C2​为任意常数)

回到本题，题目已经给出一个特解 y1=tan⁡xy_1=\tan xy1​=tanx，因此我们的唯一核心任务，就是找到第二个和 y1y_1y1​ 线性无关的特解 y2y_2y2​，而刘维尔公式，就是专门用来快速求这个 y2y_2y2​ 的固化公式，不用每次都手动降阶推导。

刘维尔降阶公式：

y2(x)=y1(x)⋅∫e−∫P(x)dxy12(x) dxy_2(x) = y_1(x) \cdot \int \frac{e^{-\int P(x)dx}}{y_1^2(x)} \, dx y2​(x)=y1​(x)⋅∫y12​(x)e−∫P(x)dx​dx

一句话总结：刘维尔公式把「求二阶方程特解」的复杂问题，直接简化成了「求一个不定积分」的简单问题，跳过了手动设函数、求导、化简、降阶的全过程。

将通解中与 y1y_1y1​ 搭配的常数，替换为关于 xxx 的未知函数 u(x)u(x)u(x)，即设第二个线性无关特解为 y2=u(x)⋅y1(x)\boldsymbol{y_2 = u(x) \cdot y_1(x)}y2​=u(x)⋅y1​(x)。
